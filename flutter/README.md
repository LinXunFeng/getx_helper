# GetX Helper

A plugin for simplifying the use of GetX

## Getting started

Add `getx_helper` to your pubspec.yaml file:


```yaml
dependencies:
  getx_helper: latest_version
```

Import `getx_helper` in files that it will be used:

```dart
import 'package:getx_helper/getx_helper.dart';
```
