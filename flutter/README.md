## 📦 安装

添加 `getx_helper` 到你的 `pubspec.yaml` 文件中:


```yaml
dependencies:
  getx_helper: latest_version
```

导入 `getx_helper` 到你要使用的文件中:

```dart
import 'package:getx_helper/getx_helper.dart';
```
